Return and Cancellation Policy

Return Policy

We want you to be completely satisfied with your purchase. If you are not satisfied, you may return the item within 30 days of receiving it for a full refund. Please ensure that the item is in its original condition and packaging.

Cancellation Policy

If you wish to cancel your order, please contact us as soon as possible. If the order has not been shipped, we will cancel it and process a full refund. If the order has already been shipped, you can still return the item according to our return policy.

How to Return an Item

To return an item, please follow these steps:

  1. Contact our customer support team at support@onlinebookstore.com to initiate the return process.
  2. Package the item securely, including all original packaging.
  3. Ship the item to the address provided by our customer support team.
  4. Once we receive the returned item, we will process the refund to your original payment method.

Contact Information

If you have any questions about our return and cancellation policies, please contact us at customer@bookstore.com.
